Online Party Game - Senior Project

Online Party Game explains itself; it is a Mario Party-inspired online board game for 2-6 players. The game's current state is fairly simple, but we left a lot of room for additions and improvments coming this summer.

Using Wisconsin (below) as a reference, I wanted to challenge myself with another online game. With the foundation set, I plan on testing the limits of real-time online connectivity within the minigames.

- Spring/Summer 2021

Wisconsin - Independent Project

Wisconsin is an online card game based off my family card game called Michigan. I created this project with my professor Dr. Nick Hwang to give my family a way to play their card game remotely during COVID-19.

Together we learned a lot about server integration in Unity. We plan on getting Wisconsin to app stores as soon as possible.

- Fall 2020/Spring 2021

End of the Party - Senior Project

End of the Party is a randomized RPG where you play as a sorcerer stopping a party of adventurers from ending you!

I created the backend database that holds and chooses scenarios for any given run held in scriptable objects. I also created the functionality of all player abilities, all UI functionality, and all the different menu systems.

- Fall 2020

Escape From Valhalla - Study Abroad

Escape from Valhalla was one of two projects created during a collaboration with UW-Whitewater and HAN University students. Unfortunately, we were unable to travel to the Netherlands, and this project was completed entirely remotely.

I was one of two programming leads for this project; I was in charge of a team of five people from both universities. My team was in charge of player mechanics and UI.
I developed the menu systems and inventory. I also created a custom input manager to support multiple controller types.

- Spring/Summer 2020

Cosmic Conflict - Mobile App
