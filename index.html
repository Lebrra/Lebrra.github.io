End of the Party - Senior Project

End of the Party is a randomized RPG where you play as a sorcerer stopping a party of adventurers from ending you!

I created the backend database that holds and chooses scenarios for any given run held in scriptable objects. I also created the functionality of all player abilities, all UI functionality, and all the different menu systems.

Split - Game Jam Continuation

Spilt is a local co-op puzzle game created during a 48 hour game jam. This project was continued, won Best in Show at the 2020 UW-Whitewater MAGD Expo, and was submitted to IndieCade!

I created the mirror effect in the background of both player screens, as well as the functionality of the title menus. I also created the endless mode, which is unlocked after getting all the collectibles in every level!
